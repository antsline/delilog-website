## 1〜2週間以内

### ⑥ X 投稿の継続（拡散フェーズ）

すでに案1を投稿済み。残り2つを段階的に投稿。

- **案2: スレッド形式（4ツイート）** — 初回投稿から2〜3日後
- **案3: 保存期間 Tips 短文** — 1週間後くらい

URLは `?v=1&utm_source=x&utm_medium=social` 形式で投稿。

**記録すること**:
- 各投稿のインプレッション・エンゲージメント・プロフィールクリック数
- どの投稿からブログへの流入が多かったか（Vercel Analytics でチェック）

### ⑦ 2本目の記事執筆

**推奨テーマ**: **「業務の記録（運行記録）6項目の書き方と保存方法」**

- 理由: 2025年4月施行の新設項目、検索需要が1年経過でピーク
- ターゲットKW: `軽貨物 業務の記録 書き方`, `運行記録 軽貨物 義務`, `業務の記録 6項目`
- 1本目から内部リンク済み（「（近日公開）」と明記している）
- 想定字数: 3,000〜4,000字

**構成案**:
1. 業務の記録が義務化された背景
2. 記録6項目の詳細と書き方
3. 「地点」の粒度（市区町村+施設名レベル）
4. リアルタイム記録の原則
5. 紙・Excel・アプリの比較
6. FAQ
7. まとめ＋CTA

### ⑧ delilog-website-5dys の削除後フォロー

- 削除後、数日間は稀に古いVercelキャッシュが動く可能性がある
- その場合 curl で確認：`curl -sI https://delilog.app/` が正常200を返すこと
- Vercel デプロイメント履歴も `delilog-website` に統一されているか確認

## 1〜3ヶ月以内（成長フェーズ）

### ⑨ 記事シリーズ完成（残2本）

- **3本目**: 「点呼記録、何を書けばいい？法令要件と具体的な記入例」
- **4本目**: 「改善基準告示まとめ：軽貨物ドライバーの勤務時間ルール」

3本以上になると相互リンクでSEO効果が指数的に上がる（トピッククラスター化）。

### ⑩ YouTube / TikTok 短尺動画（実験）

- 「3分でわかる軽貨物法令シリーズ」を1本試作
- 既存の軽貨物系YouTuberとのタイアップ打診も検討
- 反応が良ければ継続、悪ければ撤退

### ⑪ パートナーシップ開拓

以下に順番にアプローチ：

- 軽貨物マッチングプラットフォーム（PickGo、Hacobell等）のドライバー向け情報発信
- 運送業コンサル・行政書士（開業支援時にdelilog推薦）
- 軽貨物ドライバー向け LINE オープンチャット

アプローチの型：
```
こんにちは、軽貨物向けの業務管理アプリ delilog を運営しております。
最近、2025年4月施行の法令対応について記事を書きました。
もしよろしければ、ドライバーの皆さまにご紹介いただけないでしょうか。
https://delilog.app/blog/keikamotsu-anzentaisaku-2025
```

### ⑫ App Store キャンペーンパラメータ

ブログ経由のダウンロード数を計測できるようにする。

- App Store Connect → App Analytics → Campaigns
- ブログ用キャンペーンを作成
- 記事内のApp Storeリンクを `?pt=xxx&ct=blog-seo` 形式に変更
- **所要**: 30分

## 継続的な習慣（毎週・毎月）

### 毎週月曜朝（10分）

- Vercel Analytics で先週の PV・UU・Top Pages を確認
- Search Console で先週の検索キーワード・CTR を確認
- X の投稿反応を確認
- 先週のXポスト → クリック率を Vercel Analytics で突き合わせ

### 毎月末（30分）

- 月次振り返りメモ作成（別ファイルで管理推奨）
  - 公開記事数
  - 月間PV・検索流入数
  - X フォロワー・インプレッション
  - App Store DL数・有料ユーザー数
- 翌月の記事テーマを1〜2本決定
- 「伸びている記事」「伸びていない記事」の仮説分析

### KPI 目安（3ヶ月後・2026年7月末時点）

- 累計記事数: 4本
- 月間ブログPV: 500〜1,500
- Search Console 検索表示回数: 3,000〜10,000
- App Store 月間DL: 30〜50（現在の2〜3倍）
- 有料ユーザー: 3〜5人（現在0）

達成できればPMFシグナル、未達なら仮説の見直し。
