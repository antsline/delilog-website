# delilog ブログ ネタバックログ

軽貨物ドライバー向けブログ「delilog blog」の記事候補・公開済み記録・SEOメモを一元管理するファイル。

## このファイルの使い方

- **ネタを思いついたら「アイデアプール」に書き殴り**
- 書く方向が固まったら「次に書く候補」に詳細を埋めて移動
- `blog-planner` エージェントは新記事企画前にこのファイルを Read し、重複チェック・既存メモ活用
- 公開後は「公開済み」セクションへ移動
- 最終更新: 2026-04-29（候補13・14・15 追加）

## 運用ルール

### ネタ追加時
- 「アイデアプール」に1〜2行で書く
- 思いついた時点でOK、整理は後でいい

### 候補昇格時
- 「次に書く候補」に詳細項目を埋める
- 優先度（🔴 高 / 🟡 中 / 🟢 低）を付ける
- 一次資料候補を5〜10件挙げる

### 執筆開始時
- 候補のステータスを「未着手 → 企画中」に更新
- `/write-blog` 起動時、planner がこのファイルを Read して既存メモを活用

### 公開後
- 候補から「公開済み記事」セクションに移動
- 公開日・字数・反響メモを記録

### 定期メンテナンス
- 月1回: 季節カレンダーと照らし合わせて優先度を見直す
- 四半期1回: 公開済み記事の反響メモを更新
- 年1回: SEOメモのKW優先順位を見直す
